### Featured Project

#### Flight Data Platform

End-to-end platform that combines pipeline + API + AI agent:

- 6.4M flight records and 3M weather observations
- Airflow + MinIO + PostgreSQL ingestion and warehouse
- FastAPI backend with auth, caching, pagination, and pooling
- LangGraph agent with SQL tools for grounded natural-language analytics

Repo: [github.com/ManojT0108/flight-pipeline](https://github.com/ManojT0108/flight-pipeline)
